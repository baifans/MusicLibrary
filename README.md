# 2026-Spring-NKU-数据库课程设计
- 数据库：MySQL 8.0+
- 后端：JDK 21 、 Spring Boot 3.3.1 、 MyBatis 3.0.3
- 前端：Node.js 16 、 Vue 3 、 Element Plus

本系统实现了一个小型音乐库信息管理平台，主要功能模块包括：
1. 用户管理：用户注册、登录、修改个人信息（邮箱、密码）。
2. 歌曲管理：支持歌曲的增删改查。新增歌曲时由数据库触 发器自动校验（歌名非空、时长正数、同歌手 无同名歌曲）。修改歌曲仅能修改所属歌手，通过调用存储过程sp_change_song_singer 完成，该过程包含事务、存在性检查和同名检查。删除歌曲通过存储过程 sp_delete_song 级联删除歌单关联和评论，保证数据完整性。
3. 歌手管理：支持按歌手名模糊查询，删除歌手时调用存储过程sp_delete_singer，事务内级联删除该歌手的歌曲、评论、歌单关联，并对热门歌手（歌曲数≥10）进行保护。
4. 评论管理：展示所有评论，内容包含评论内容、用户名、歌曲名。
5. 歌单管理：展示歌单名称和创建者，支持点击“详情”弹窗显示歌单内所有歌曲（歌曲名、歌手、时长）。
6. 热歌榜单：基于视图 v_hot_song（播放量 5000 的歌曲），支持按歌曲名、歌手名查询 。
